# seeds

任意で投入する初期データ（サンプル）置き場です。マイグレーション（`migrations/`）とは別で、
**運用開始の利便のために1回だけ流す**ものです。phpMyAdmin の「SQL」タブで実行します。

| ファイル | 内容 |
|---|---|
| `training_items_sample.sql` | 研修マスターの標準項目（対象カラー別・共通）20件。`tenant_id=1` に既存データがあれば**何も挿入しない**ガード付き（再実行安全）。 |

## training_items_sample.sql

- 対象カラー（GREEN/BLUE/YELLOW/RED）ごとに、Color育成制度の役割に沿った標準項目を投入。
- すべて **部門＝空（共通）** で、全講師に表示されます。
- 投入後、「研修マスター」画面で自由に編集・並び替え・**部門別項目の追加**ができます。
- あくまで**たたき台**です。実際の研修体系に合わせて調整してください。

> マイグレーション（`001`〜`003`）を適用済みであることが前提です（`training_items` テーブルが必要）。
