1. はじめに（システム構成）

本システムは、ひとつのログインで使える 2つのアプリ で構成されています。

アプリ役割主な利用者
ColorHRM
/colorhrm/		採用・育成（研修の進捗・承認、講師情報、カラー制度 WHITE→GREEN→BLUE→YELLOW→RED）講師・スタッフ・管理者
給与・シフト
/colorhrm-pay/		打刻・シフト・給与計算・給与明細講師・スタッフ・管理者

2つのアプリは、各画面の上部・メニューにある相互リンクで行き来できます。アカウント（ログインID・パスワード）は共通です。

権限について
・講師（teacher）… 自分のマイページ・打刻・シフト・給与明細を利用します。
・スタッフ（staff）… 自分の担当教室の講師について、育成・研修承認・シフト・給与を管理します。
・管理者（admin）… すべての機能（マスター管理・ユーザー管理を含む）を利用できます。

2. 共通操作

2.1 ログイン

  1. ブラウザでアプリのURL（例：https://chishokan.co.jp/colorhrm/）を開きます。
  2. メールアドレス（ログインID）とパスワードを入力し、ログインします。
  3. 講師でログインすると自動的にマイページが開きます。
ログインIDとパスワードは管理者から発行されます。不明な場合は管理者にお問い合わせください。

2.2 2つのアプリの行き来

ColorHRM 画面からは「💴 給与・シフト」、給与・シフト画面からは「🎓 ColorHRM」のリンクで移動できます。再ログインは不要です。

2.3 スマートフォンでの利用

スマホでも利用できます。スタッフ画面では左上の 三本線（≡） をタップするとメニュー（サイドバー）が開きます。講師画面では上部にボタンが並びます。

2.4 ログアウト

共有端末を使う場合は、画面右上の「ログアウト」で必ずログアウトしてください。

2.5 ヘルプ（アプリ内）

各アプリのメニューにある ヘルプ・使い方 から、いつでも最新の操作説明を確認できます。

3. 講師（teacher）向け

3.1 ColorHRM マイページ

ログイン後の最初の画面です。次のことができます。

  • 自分の現カラー／育成目標カラーと達成率の確認
  • 研修進捗の確認と提出・報告
  • 顔写真の登録
  • 管理者からの質問への回答

研修進捗の進め方

研修項目には種別タグが付いています。種別ごとに操作が異なります。

種別操作
テスト OJT テスト結果・OJT記録の写真を選び、写真提出（OJTは OJT写真提出）を押す。JPG/PNG・上限8MB。
研修 LINE WORKS にリフレクションを投稿してから 投稿済み を押す（写真は不要）。

ステータスの流れ

未着手 →（提出・報告）→ 申告中（承認待ち）→ 担当者が承認 → 合格／受講済

差戻し・不合格 になった場合は、内容を直してもう一度提出・報告してください。必須 の項目はカラー昇格に必要です。📺 教材 がある項目は、押すと動画・資料を確認できます。

顔写真の登録／質問への回答

  • マイページ左上の写真欄からファイルを選び「写真を登録」（JPG/PNG・上限8MB）。
  • 管理者からの質問はマイページ下部に表示されます。回答を入力して「保存」（後から修正可）。

3.2 給与・シフト

打刻（出勤・退勤）

  1. メニューの黄色い 打刻 を開きます。
  2. 「出勤」欄で出勤教室を選び 出勤打刻。
  3. 退勤時は「退勤」欄で退勤教室を選び 退勤打刻。

シフトがない日でも打刻できます。その場合は判定が シフトなし になります。

判定意味
OK問題なし
遅刻シフト開始より遅い出勤
早退シフト終了より早い退勤
欠勤打刻なし
シフトなしその日のシフトが無い
配属教室が未設定だと打刻できません。教室が選べない場合は、管理者・スタッフに配属教室の設定を依頼してください。

シフト可能登録

  • 入れる日だけ開始・終了の時間を入力します（空欄の日は申請しません）。
  • 「↑ 上をコピー」で前の行の時間をコピーできます。
  • 入力したら 保存する を押します。当月〜6か月先まで登録できます。
  • スタッフが確定した行は編集できなくなります（画面下部の一覧で確認）。

給与明細

「給与明細」から発行済みの明細を確認し、PDFを開く で表示・保存できます。金額は発行時点の確定シフトに基づきます。

4. 職員・スタッフ（staff）向け

スタッフ権限では、自分の担当教室に配属された講師が表示・管理の対象になります（担当教室の設定は管理者が行います）。

4.1 ColorHRM（採用・育成）

ダッシュボード

  • 研修 承認待ち（教室別）… 講師の申告を教室ごとに一覧。「承認へ」でその講師の研修管理に移動。
  • 育成サマリー… カラー別人数、目標期限超過・達成率の低い要注意リスト、教室×カラー分布。

講師一覧・講師の追加

  • 「講師一覧」で在籍講師を確認。＋ 講師を追加 で新規登録し、続けて詳細を設定します。
  • 講師名や「研修進捗」から、その講師の詳細・研修チェックリストへ移動できます。

講師詳細でできること

  • プロフィール編集（氏名・メール＝ログインID・部門・校舎・メンター 等）→「プロフィールを保存」
  • 配属教室の設定（打刻に必須・複数選択可）
  • カラー昇格／退職・復職の切り替え／1on1 面談の記録

研修管理（承認の手順）

  1. 「研修管理」を開くと 承認待ち の申告が一覧されます。
  2. 講師・項目・種別・メモ・写真（写真 を押すと表示）を確認します。
  3. 結果（合格／受講済／不合格）を選び 承認。不備があれば 差戻し。

講師名を押すと進捗グリッドが開き、各項目のステータスを直接「更新」することもできます。

4.2 給与・シフト

左サイドメニューは ダッシュボード／シフト（シフト表・シフト管理）／給与（給与計算）に分かれています。

ダッシュボード

在籍講師数・給与計算対象数・当月の確定シフト（前月/翌月へ移動可）・講師別の時給を確認できます。

シフト表

教室を選ぶと、その教室の講師×日付で 申請中 と 確定 を一覧できます。

シフト管理（確定の手順）

  1. 申請中の各行で 教室 と 授業（分） を入力し 確定（不要なら 却下）。
  2. まとめて確定する場合は この月をまとめて確定。
  3. 申請がないシフトは「確定シフトを直接追加」から登録。
  4. 確定シフトの一覧では、教室・時間・授業分の修正（「保存」）や削除ができます。

講師は1日に複数の教室で勤務することがあります。日ごとに確定した教室が、その教室のシフト表・講師のマイページに反映されます。

給与計算・明細発行

  • 月を選ぶと、確定シフトと時給表から自動計算されます。
  • 発行（または「全員に発行」）で明細を発行。
  • PDF で明細PDFを確認、メール送信 で講師へ送付（PDF出力とメール送信は分かれています）。
  • CSVダウンロード／コピーで表計算ソフトに貼り付けできます。
計算の考え方：授業給与＝授業分÷60×授業時給、運営給与＝運営分÷60×運営時給、交通費は勤務日数で算定。 金額のもとになる勤務時間は確定シフトの時間です（打刻の時刻は遅刻・早退・欠勤の判定にのみ使い、給与額には反映しません）。

5. 困ったときは（FAQ）

Q. ログインできない／パスワードを忘れた

管理者にお問い合わせください。パスワードの再発行ができます。

Q. 打刻で教室が選べない

配属教室が未設定です。スタッフ・管理者に、ColorHRMの講師詳細での配属教室の設定を依頼してください。

Q.（スタッフ）担当の講師が表示されない

その講師の配属教室と、あなたの担当教室が一致しているか管理者にご確認ください。

Q. シフトを登録したのに給与に反映されない

給与は確定したシフトから計算されます。スタッフがシフト管理で確定しているかご確認ください。

Q. 研修を提出したのに合格にならない

提出後は 申告中（承認待ち）です。スタッフ・管理者の承認をお待ちください。差戻しの場合は再提出してください。

最新の操作説明は、各アプリのメニュー「ヘルプ・使い方」からも確認できます。
